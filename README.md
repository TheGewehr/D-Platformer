## GAME CONTROLS
~~~~~~~~~~~~~~~

Keyboard Player controls:
- JUMP --> W key
- LEFT --> A key
- RIGHT --> D key

General controls:
- SKIP SCREEN --> SPACE key
- EXIT GAME --> ESC key
- RESTART GAME --> X key

Debug Controls:
- SAVE GAME --> S key
- LOAD GAME --> L key
- LEVEL 1 --> F1 key
- LEVEL 2 --> F2 key (Not ready yet)
- WIN INSTANT --> F3 key
- LOSE INSTANT --> F4 key
- SHOW CHAINS --> F9 key
